# Student Management System

## About the Project

This is a simple console-based Student Management System developed in C++. The project was created to practice file handling, structures, vectors, and basic CRUD (Create, Read, Update, Delete) operations.

The program stores student records in a text file so that the data remains available even after the program is closed.

## Features

* Add new student records
* View all students
* Search a student using roll number
* Update student details
* Delete student records
* Display total number of students
* Store data permanently using file handling

## Technologies Used

* C++
* File Handling
* Structures
* Vectors
* Standard Template Library (STL)

## File Structure

student_management.cpp - Main source code

students.txt - Stores student records

.gitignore - Ignores unnecessary files

## How to Run

Compile the program:

g++ student_management.cpp -o sms

Run the program:

./sms

(On Windows, run `sms.exe` after compiling.)
